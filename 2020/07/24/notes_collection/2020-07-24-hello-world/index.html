To Fix Some Bugs 🐬

  • hexo s 命令失效
    某人将 hexo 整个文件夹上传到 GitHub 服务器，在另一台电脑下载下来，运行 hexo s 命令失效，从显示信息得知，不存在该命令。这可能是由于 Hexo3 将 hexo-server 独立出来，如果需要本地调试，需要先安装 server。
